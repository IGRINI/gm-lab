# GM-Lab — дизайн-система веб-UI

Стиль v3 «modern floating dark» (2026-07-12). Холодный почти-чёрный нейтрал,
один лавандовый акцент, пилюли и большие радиусы, «острова» вместо полос с
рамками, светлые активные элементы. Всё живёт в `web/src/theme.css`
(токены + примитивы), остальные файлы стилей только потребляют токены.

## Правила (нарушать нельзя)

1. **Острова, не полосы.** Хедер и сайдбар прозрачные на канвасе; панели
   (HUD, карточки, модалки) — скруглённые острова `surface-1` с мягкой тенью
   `--shadow-1/2/3`. Полос с `border-bottom` на всю ширину не делаем.
2. **Активное состояние = светлая пилюля** (`--active-pill` + `--active-ink`),
   не цветная заливка. Акцент `--brand` (лаванда #cbbcf6, светлый solid с
   тёмным текстом `--brand-ink`) — только у главных CTA и фокуса.
3. **Радиусы крупные**: `--r-sm 10 / --r-md 14 / --r-lg 18 / --r-xl 24 /
   --r-full`. Кнопки — пилюли (`.btn`), иконочные — тайлы 36×36 `r-sm`
   (`.icon-btn`). Прямых углов нет.
4. **Один гротеск** — Inter Variable для всего (`--font-book` тоже указывает
   на Inter; серифов в UI нет). Размеры: 11 кикер / 12 подпись / 13 дефолт /
   14 заголовок карточки / 16 заголовок секции / 19-22 титулы; крупные
   заголовки — вес 650 + letter-spacing -0.01em.
5. **Ховер светлее** (surface-2 → surface-3), фокус — единое кольцо
   `:focus-visible` лавандой. Переходы 120/200мс `--ease`.
6. **Никаких эмодзи как иконок** — только `components/Icon.jsx` (24×24,
   stroke 1.8). Новую иконку добавлять в словарь, не инлайнить SVG по месту.
7. Числа, меняющиеся на месте (HP, токены) — `tabular-nums`.
8. Легаси-алиасы (`--bg/--bg2/--bg3/--line/--tx/--mut/--sub/--acc/--redo`)
   работают, но в новом коде использовать семантические токены.

## Голоса транскрипта

- Наррация ГМ — чистый текст без карточки, тихий кикер «ГЕЙМ-МАСТЕР» (text-3).
- Игрок — мягкий голубоватый пузырь справа без рамки (r-18, хвостик 6).
- NPC — остров с тинтом цвета персонажа (color-mix 7% + surface-1, r-18),
  имя цветом, без рубчиков.
- Тул-карты — остров surface-1, цвет инструмента только в тайле-иконке (--tc).
- Кости (3D-арт) не трогать — физический реквизит, золото уместно.

## Известные ограничения стенда

- Мок-превью (`--server` + `GM_BACKEND=mock`) отдаёт `/transcript` голым массивом;
  App принимает обе формы. В скрытой вкладке (headless-панель) rAF заморожен —
  live-лента не обновляется, это ограничение браузера, не бага UI.
